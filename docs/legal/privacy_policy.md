# 隐私协议（隐私政策）

> 英文标题：**AgenticX / Machi Privacy Policy**

**更新日期：2026年04月**
**生效日期：2026年04月**

欢迎使用 AgenticX / Machi 桌面 AI 助手（以下简称“本软件”）。本软件由个人开发者 Damon Li（以下简称“开发者”或“我们”）开发并提供。

作为一款强调“本地优先（Local First）”的生产力工具，本软件的核心设计理念之一就是最大程度保障您的数据隐私。我们深知个人信息和本地文件对您的重要性，因此我们制定了本《隐私政策》以帮助您了解：在您使用本软件时，我们会涉及哪些信息的处理？这些信息存放在何处？以及我们如何保护您的隐私。

**核心摘要：**
- **您的文件、日志、记忆和配置默认均保存在您本地设备上（`~/.agenticx` 目录）。**
- **我们（开发者）没有中央服务器来收集您的代码、工作区文件或聊天记录。**
- **您的对话内容和相关工作区文件将通过网络发送给您所配置的第三方大语言模型（LLM）提供商（如 OpenAI、Anthropic、智谱等）进行推理生成。**

请您在使用前仔细阅读并确认理解本隐私政策。

---

## 1. 我们如何收集和使用您的信息

### 1.1 您的本地工作区文件与代码
由于本软件具备文件读取、代码审查和项目分析功能，当您在聊天中提及文件（如 `@文件`）或授权分身（Avatar）和工具操作时，本软件会自动读取这些本地文件（包括代码、文档、配置等）的内容。
**【如何使用】**：这些内容仅用于拼接成上下文（Context）或提示词（Prompt），并发送给您所选择的 LLM 进行推理。
**【数据去向】**：这些内容**不会**上传至开发者的服务器，而是**直接加密传输**至您配置的模型提供商（见第 2 节）。

### 1.2 对话记录与记忆（Memory）
您在使用过程中产生的所有对话记录（Chat History）、用户偏好（User Preferences）和自动生成的长短期记忆（Memory）。
**【数据去向】**：保存在您本机的 `~/.agenticx` 目录及 SQLite 数据库中。我们（开发者）无法查看、收集或获取这些信息。

### 1.3 第三方 API 密钥及账号绑定
如果您在设置中配置了第三方 API Key（例如 OpenAI 的密钥），或绑定了第三方应用（如微信、飞书、Figma 等）。
**【数据去向】**：这些密钥和凭证将加密并仅保存在您本地设备的配置文件中。除了作为授权凭证向对应的第三方接口发起正常请求外，它们**绝不会**被发送至任何其他服务器。

### 1.4 遥测与诊断数据（可选）
在未来版本中，为了改进产品，我们可能会请求收集非常基础的匿名使用数据（例如崩溃日志、应用启动次数、遇到错误的频率）。如果引入该机制，我们将提供明确的“开启/关闭”选项，并确保绝对不会包含您的聊天正文、代码或敏感数据。当前版本下，**我们不主动收集任何诊断数据**。

## 2. 信息共享与第三方服务

本软件本身不存储您的任何云端用户数据，但本软件依赖于外部 AI 提供商来为您提供智能服务。因此，**您的提示词和文件内容将被发送给第三方**。您在此过程中的隐私安全受这些第三方隐私政策的约束：

### 2.1 第三方大语言模型（LLM）API
当您使用基于云端的模型（如 OpenAI GPT-4、Anthropic Claude 3.5、智谱 GLM-4 等）时，您的提问、工作区上下文、甚至终端错误日志都会被发送至该服务商的 API。
*注：大多数主流模型服务商的 API 隐私条款均声明，通过 API 提交的数据不会用于训练其公共模型（请以该服务商的最新条款为准），但数据仍会流经其服务器。*
- 您可以随时在软件设置中切换模型提供商。
- 如果您对隐私要求极高，可以在设置中选择接入本地部署模型（如 Ollama），在此模式下，数据将完全在您的本地网络内流转，不会出网。

### 2.2 外部工具与 MCP (Model Context Protocol) 插件
当本软件利用工具进行网络搜索（Web Search）、网页内容抓取（Crawler）或接入第三方 MCP 服务（如 GitHub、Figma）时，相关的查询参数和数据将会发送给对应的服务方。

## 3. 我们如何存储和保护您的信息

1. **本地存储**：您的数据主要保留在本地设备。保护您设备免遭非法访问是确保数据安全的关键，建议您妥善设置操作系统的密码及安全防护。
2. **传输安全**：所有向第三方 API 发送的请求均通过 HTTPS/TLS 加密传输。
3. **数据控制权**：您对您的数据拥有完全控制权。您可以随时清空聊天记录、删除分身（Avatar）配置，或直接删除本地的 `~/.agenticx` 文件夹来彻底清除所有软件数据和痕迹。

## 4. 未成年人保护

本软件主要面向成年专业用户（如开发者、研究人员等）。我们不面向14周岁以下的未成年人提供服务，也不会有意收集未成年人的个人信息。如果您是未成年人，请在监护人陪同下停止使用本软件。

随着产品功能的迭代或法律法规的变更，我们可能适时修订本《隐私政策》。当隐私政策发生重大变更时，我们会在软件内或官网发布最新版本，请您在使用前随时关注。如果您继续使用本软件，即表示同意接受修订后的隐私政策。
